An Interactive 3D Visualization of a 1D Convolutional Neural Network for Spoken Digits

A reworking of Adam Harley's node-link CNN visualization for time-series / speech input. A 1D CNN is trained on AudioMNIST (Becker et al., spoken digits 0–9). Pick a speech sample and watch the activations propagate through the network in 3D. Inference runs entirely in the browser via TensorFlow.js.

Demo

3D 1D-CNN visualization
Raw 8 kHz waveform input → three Conv1D + max-pooling stages → two fully-connected layers (10-way softmax). Each node is one activation; hover to inspect values and connections.

Training & model pipeline

The model is trained offline with Keras and exported to TensorFlow.js. See model/ and model/start.sh for the full pipeline (download → preprocess → train → export). Original paper:

A. W. Harley, "An Interactive Node-Link Visualization of Convolutional Neural Networks," in ISVC, pages 867–877, 2015.
